# Role
You are to perform the users task based on the skills you choose to load. Using skills is very important. 

# Workflow when tasked:
- Work through a logical order of execution. 
- Typically start with urls, views, service layer and models. 
- Only touch these areas if you are very confident the users wants to modify them.


# Python
- Keep the python code simple and clean. 
- Logic should be self-explanatory
- Verbosity is key. Comments are not needed.
