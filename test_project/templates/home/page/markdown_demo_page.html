Markdown Template Tag Demo

This page demonstrates the {% verbatim %}{% render_markdown %}{% endverbatim %} and {% verbatim %}{% render_markdown_stripped %}{% endverbatim %} template tags.

Basic Render

Code: {% verbatim %}{% render_markdown "test_project/docs/sample_readme.md" %}{% endverbatim %}

{% render_markdown "test_project/docs/sample_readme.md" %}
As Context Variable

Code: {% verbatim %}{% render_markdown "test_project/docs/sample_readme.md" as md %}{% endverbatim %}
{% verbatim %}{% if md %}{{ md|safe }}{% endif %}{% endverbatim %}

{% render_markdown "test_project/docs/sample_readme.md" as md %} {% if md %}
Success! File was found and rendered below:

{{ md|safe }}
{% else %}
File not found
{% endif %}
Stripped (Plain Text)

Code: {% verbatim %}{% render_markdown_stripped "test_project/docs/sample_readme.md" as stripped %}{% endverbatim %}

Useful for previews, search, or excerpts without HTML formatting.

{% render_markdown_stripped "test_project/docs/sample_readme.md" as stripped %} 
{{ stripped|truncatewords:50 }}
Missing File (Graceful None)

Code: {% verbatim %}{% render_markdown "docs/does_not_exist.md" as missing %}{% endverbatim %}

When a file doesn't exist, the tag returns None, allowing for graceful fallbacks.

{% render_markdown "docs/does_not_exist.md" as missing %} {% if missing %} {{ missing|safe }} {% else %}
No file found — The file docs/does_not_exist.md does not exist. This is the expected behavior when a file is missing.
{% endif %}
